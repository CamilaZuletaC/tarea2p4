¿Que es?


Los condicionales son el uso de funciones preestablecidas y el uso de restricciones para que estas operen. Los condicionales se pueden utilizar para modificar el comportamiento de todas las funciones que utiliza excel

Nos enseñaron a utilizar el condicional de suma. Este se utiliza para sumar todo los valores, o un conjunto de numeros especificados por el usuario, un ejemplo para este seria: Se quiere sumar de un conjunto de numeros, las notas buenas(superiores a tres), los numeros del conjunto son: 4,3,5,1,2,4,2,4,1,2.En este caso seria, primero se debe poner el nombre de la operacion, en este caso suma, siguiente a esto se pone un punto y un si,se abre parentesis y se debe colocar el rango(donde quiere buscar la condicion), entonces el rango de busqueda son todos los numeros del conjunto, luego sigue el criterio; ud que quiere(mayores,igual que 3), luego rango de la suma(se suma las mismas notas).
